Glossary

Meta Elements
Meta elements are HTML tags that provide information about the page and are placed in the 'head' section of a HTML document.
favicons
Favicons are small images on the tab of your webpage that can be used to distinguish your site.
-Br- element
The br element is used to insert a line break in a text.
-Hr- element
The hr element is used to create a line between paragrahs.
-cite- element
The cite element is used to define the title and author of a work.
